# Plataforma de Monitoreo y Análisis de Ataques en Redes IP

Este proyecto consiste en desarrollar una plataforma de monitoreo de ataques en redes IP utilizando Python, Scapy, Wireshark y Flask.

## Requisitos

- Python 3.8+
- VirtualBox/VMware
- Wireshark
- Scapy
- Flask
